Motivation:
* Q4: Purpose of the creation of dataset?
Q4-Ans: To model and analyze social-network interactions with typed, time-stamped edges (hasMember, knows, likeComment, like_post), affiliations via workAt (with workFrom), and geolocation via isLocatedIn and isPartOf in a property graph.

* Q5: Which task was intended to solve with this dataset, i.e., data mining, Q/A, entity linking, path query etc?
Q5-Ans: Graph querying and mining (path queries, pattern matching), social network analysis, recommendation/link prediction, and entity linking via DBpedia URIs.
